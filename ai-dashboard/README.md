# AI TechStack Dashboard App

This folder contains the runnable React + Vite application for the AI TechStack repository.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Main Source Files

- `src/App.jsx`: role data, roadmap levels, and UI
- `src/main.jsx`: React entry
